🔤 Las Variables

Las variables son contenedores que almacenan valores. En JavaScript existen 3 tipos principales de variables, cada una con un propósito y alcance (scope) diferente.

let

Es una variable con alcance de bloque (block scope).

✓ Alcance: Solo dentro del bloque donde se declara (if, for, while, etc.)

✓ Reasignación: SÍ se puede cambiar su valor

✓ Redeclaración: NO en el mismo bloque

let edad = 25;
edad = 26;

var

Es una variable con alcance de función (function scope).

⚠ Alcance: Todo el programa/función, independientemente de bloques

✓ Reasignación: SÍ se puede cambiar su valor

✓ Redeclaración: SÍ se puede redeclarar

var nombre = "Juan";
var nombre = "Carlos";

const

Una constante, su valor NO cambia (inmutable).

🔒 Alcance: Solo dentro del bloque donde se declara

✗ Reasignación: NO se puede cambiar su valor

✗ Redeclaración: NO se puede redeclarar

const PI = 3.14159;
PI = 3.14; // ❌ Error

📊 Tabla Comparativa

Característica let var const
Alcance Bloque Función Bloque
Reasignable ✓ Sí ✓ Sí ✗ No
Redeclarable ✗ No ✓ Sí ✗ No
Hoisting TDZ* undefined TDZ*

*TDZ = Temporal Dead Zone (zona muerta temporal)

💡 Recomendación: Usa const por defecto, let cuando necesites reasignar, y evita var en código moderno.

🔗 Concatenación de Cadenas

La concatenación es el proceso de unir dos o más cadenas de texto en una sola. Existen varias formas de hacerlo en JavaScript.

1️⃣ Usando el Operador +

Es la forma más básica y tradicional.

let nombre = "Juan";
let saludo = "Hola, " + nombre + "!";
console.log(saludo); // "Hola, Juan!"

2️⃣ Usando el Método concat()

Método específico para concatenar cadenas.

let saludo = "Hola".concat(", ", "Juan", "!");
console.log(saludo); // "Hola, Juan!"

3️⃣ Usando Template Literals (Backticks)

La forma más moderna y flexible. Permite expresiones dentro de ${}.

let nombre = "Juan";
let edad = 25;
let saludo = `Hola, ${nombre}. Tienes ${edad} años.`;
console.log(saludo); // "Hola, Juan. Tienes 25 años."

4️⃣ Usando append() en Arrays

Convertir a string y unir elementos.

let partes = ["Hola", "Juan"];
let saludo = partes.join(" ");
console.log(saludo); // "Hola Juan"

💡 Recomendación: Usa template literals (`${}`) para una mejor legibilidad, especialmente cuando tengas múltiples variables.

📦 Arrays y Objetos

Los arrays y objetos son estructuras de datos que permiten almacenar múltiples valores de forma organizada.

Arrays (Listas)

Un array es una colección ordenada de elementos. Se accede mediante índices (posiciones).

let frutas = ["manzana", "plátano", "naranja"];
console.log(frutas[0]); // "manzana"
console.log(frutas.length); // 3

Objetos (Diccionarios)

Un objeto es una colección de pares clave-valor. Se accede mediante el nombre de la propiedad.

let pc = {
  nombre: "PCJuan",
  procesador: "Intel Core i7",
  ram: "32GB",
  almacenamiento: "2TB"
};

console.log(pc.nombre); // "PCJuan"
console.log(pc["procesador"]); // "Intel Core i7"

Diferencias Clave

Característica Array Objeto
Acceso Por índice numérico (0, 1, 2...) Por nombre de propiedad
Orden Ordenado No siguiendo orden específico
Mejor para Listas, colecciones Agrupar propiedades relacionadas

🔄 Bucles e Iteración

Los bucles permiten ejecutar un bloque de código repetidas veces, de manera automática, lo que ahorra código y facilita el manejo de colecciones de datos.

WHILE

Uso: Se repite mientras la condición sea verdadera.

let i = 0;
while (i < 5) {
  console.log(i);
  i++;
}

DO WHILE

Uso: Ejecuta primero, luego pregunta. Mínimo una ejecución.

let i = 0;
do {
  console.log(i);
  i++;
} while (i < 5);

FOR

Uso: Bucle determinado con inicio, condición e incremento.

for (let i = 0; i < 5; i++) {
  console.log(i);
}

Partes: inicialización ; condición ; incremento

FOR IN

Uso: Recorre las propiedades de un objeto o índices de un array.

let obj = {a: 1, b: 2};
for (let key in obj) {
  console.log(key); // "a", "b"
}

FOR OF

Uso: Recorre los valores de un array o iterable.

let frutas = ["manzana", "plátano"];
for (let fruta of frutas) {
  console.log(fruta);
}

FOREACH

Uso: Método que ejecuta una función en cada elemento.

frutas.forEach((fruta, index) => {
  console.log(index, fruta);
});

Sentencias de Control en Bucles

BREAK

Detiene completamente el bucle.

for (let i = 0; i < 10; i++) {
  if (i === 5) break;
  console.log(i);
}

CONTINUE

Salta la iteración actual.

for (let i = 0; i < 10; i++) {
  if (i === 5) continue;
  console.log(i);
}

LABEL

Etiqueta un bucle para control específico.

miEtiqueta: for (...) {
  break miEtiqueta;
}

🔧 Funciones

Las funciones son bloques de código reutilizables que realizan tareas específicas. Permiten organizar el código, evitar repeticiones y hacerlo más mantenible.

Declaración de Función

function saludar() {
  console.log("¡Hola!");
}

saludar(); // Llamar la función

Funciones con Parámetros

Los parámetros permiten que las funciones reciban datos y se comporten de manera dinámica.

function saludar(nombre) {
  console.log(`¡Hola, ${nombre}!`);
}

saludar("Juan"); // ¡Hola, Juan!

RETURN - Devolver Valores

El return devuelve un valor que puede usarse en otra parte del código.

function sumar(a, b) {
  return a + b;
}

let resultado = sumar(5, 3);
console.log(resultado); // 8

Funciones Flecha (Arrow Functions)

Forma más concisa e moderna de declarar funciones.

// Forma larga
const sumar = (a, b) => {
  return a + b;
};

// Forma corta
const sumar = (a, b) => a + b;

Función Anónima

Función sin nombre, típicamente asignada a una variable.

const multiplicar = function(a, b) {
  return a * b;
};

Parámetros Predeterminados

function saludar(nombre = "Amigo") {
  console.log(`¡Hola, ${nombre}!`);
}

saludar(); // ¡Hola, Amigo!

Comparación de Sintaxis

Tipo Sintaxis Ventajas
Función Normal function name() {} Hoisting, legible, traditional
Función Expresión const name = function() {} Puede asignarse sin hoisting
Arrow Function const name = () => {} Sintaxis corta, modern, mejor `this`

🏗️ Programación Orientada a Objetos (POO)

La POO es un paradigma que organiza el código en objetos con propiedades (datos) y métodos (funciones). Facilita crear sistemas complejos y mantenibles.

Conceptos Fundamentales

📋 Clase

Es el plano o molde para crear objetos.

class Persona {
  constructor(nombre) {
    this.nombre = nombre;
  }
}

🔧 Constructor

Método especial que se ejecuta al crear un objeto.

let maria = new Persona("María");

📦 Propiedades

Los datos que caracterizan al objeto.

this.nombre = "Juan";
this.edad = 25;

⚙️ Métodos

Funciones que asocian comportamiento.

saludar() {
  return `Hola, soy ${this.nombre}`;
}

🔐 Encapsulamiento

Control de acceso a propiedades.

#saldo = 0; // privado

👨‍👧 Herencia

Una clase hereda de otra.

class Estudiante extends Persona {}

🎭 Polimorfismo

El mismo método se comporta diferente.

Métodos con mismo nombre, distinto comportamiento.

📚 Métodos de Arrays

Los arrays tienen métodos muy útiles para manipular, transformar y buscar datos.

Métodos Modificadores (Transformadores)

Estos métodos cambian el array original:

push()

Agrega elemento(s) al final.

[1,2].push(3); // [1,2,3]

pop()

Elimina el último elemento.

[1,2,3].pop(); // [1,2]

shift()

Elimina el primer elemento.

[1,2,3].shift(); // [2,3]

unshift()

Agrega elemento(s) al inicio.

[2,3].unshift(1); // [1,2,3]

reverse()

Invierte el orden.

[1,2,3].reverse(); // [3,2,1]

sort()

Ordena alfabéticamente.

[3,1,2].sort(); // [1,2,3]

splice()

Elimina y/o agrega elementos.

[1,2,3,4].splice(1,2); // [1,4]

Métodos de Acceso (No modifican)

Estos métodos NO cambian el array original:

slice()

Copia parte del array.

[1,2,3,4].slice(1,3); // [2,3]

concat()

Une dos arrays.

[1,2].concat([3,4]); // [1,2,3,4]

join()

Convierte a string con separador.

[1,2,3].join("-"); // "1-2-3"

indexOf()

Busca el índice del elemento.

[1,2,3].indexOf(2); // 1

includes()

Verifica si contiene elemento.

[1,2,3].includes(2); // true

Métodos de Iteración

Ejecutan funciones en cada elemento:

forEach()

Ejecuta función en cada elemento.

[1,2,3].forEach(n => console.log(n));

map()

Crea array con resultados.

[1,2,3].map(n => n*2); // [2,4,6]

filter()

Filtra por condición.

[1,2,3,4].filter(n => n>2); // [3,4]

find()

Encuentra el primer elemento.

[1,2,3].find(n => n>2); // 3

reduce()

Reduce array a un valor.

[1,2,3].reduce((a,b) => a+b); // 6

some()

¿Alguno cumple la condición?

[1,2,3].some(n => n>2); // true

every()

¿Todos cumplen la condición?

[1,2,3].every(n => n>0); // true

🔤 Métodos de Cadena

Los métodos de cadena permiten manipular strings de diversas formas.

concat()

Une dos o más cadenas y retorna una nueva.

"Hola".concat(" ", "Mundo"); // "Hola Mundo"

startsWith()

Verifica si una cadena comienza con ciertos caracteres.

"JavaScript".startsWith("Java"); // true

endsWith()

Verifica si una cadena termina con ciertos caracteres.

"JavaScript".endsWith("Script"); // true

includes()

Verifica si una cadena contiene otra.

"Hola Mundo".includes("Mundo"); // true

indexOf()

Retorna el índice de la primera ocurrencia.

"Hola".indexOf("l"); // 2 (-1 si no existe)

lastIndexOf()

Retorna el índice de la última ocurrencia.

"Hola".lastIndexOf("l"); // 3

slice()

Extrae una parte de una cadena.

"Hola".slice(0, 3); // "Hol"

substring()

Similar a slice() pero sin soportar índices negativos.

"Hola".substring(0, 3); // "Hol"

toLowerCase()

Convierte a minúsculas.

"HOLA".toLowerCase(); // "hola"

toUpperCase()

Convierte a mayúsculas.

"hola".toUpperCase(); // "HOLA"

trim()

Elimina espacios en blanco al inicio y fin.

" Hola ".trim(); // "Hola"

split()

Divide una cadena en un array.

"a,b,c".split(","); // ["a", "b", "c"]

replace()

Reemplaza la primera ocurrencia.

"Hola Hola".replace("Hola", "Hi"); // "Hi Hola"

repeat()

Repite una cadena N veces.

"abc".repeat(3); // "abcabcabc"

padStart()

Rellena al principio hasta cierta longitud.

"5".padStart(3, "0"); // "005"

padEnd()

Rellena al final hasta cierta longitud.

"5".padEnd(3, "0"); // "500"

Math.abs()

Valor absoluto.

Math.abs(-5); // 5

Número mayor.

Math.max(1,5,3); // 5

Número menor.

Math.cos(0); // 1

🌐 El DOM (Document Object Model)

El DOM es la representación en árbol de tu página HTML. Con JavaScript puedes acceder, modificar y manipular elementos del DOM.

¿Qué es el DOM?

El DOM convierte tu HTML en objetos que JavaScript puede manipular. Cada etiqueta HTML es un "nodo".

<html>
  <head>...</head>
  <body>
    <h1>Título</h1>
    <p>Párrafo</p>
  </body>
</html>
